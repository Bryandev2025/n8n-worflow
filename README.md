# modern-vite-react-site

A modern, accessible, and production-ready React site scaffolded with Vite.

Features
- React 18+ with Vite
- React Router (Home & About)
- Theme toggle (light/dark) persisted with localStorage
- Accessible header with skip link
- Responsive layout and utility classes
- Framer Motion animations
- Axios for placeholder data fetching in Features
- Contact form with validation and simulated submission
- ESLint and Prettier configuration

Quick start
1. Install dependencies:
   npm install

2. Run development server:
   npm run dev

3. Build for production:
   npm run build

4. Preview production build:
   npm run preview

Lint & format
- Lint: npm run lint
- Format: npm run format

Project structure
- src/
  - assets/
  - components/
  - hooks/
  - pages/
  - utils/
